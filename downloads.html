The current version of libmapper is version 0.3, and can be downloaded here. While we are in the process of building binaries for 0.3, we recommend that you build the library from source. It is licensed with the LGPL v2.1 or later. The full list of files, including any previous releases, can be found at http://idmil.org/pub/software/libmapper.
