Videos of works using libmapper

2022

Mapper4Live: Using Control Structures to Embed Complex Mapping Tools into Ableton Live

Mapper4Live is a Max for Live plugin that allows for complex connections between gestural signals and Ableton Live parameters. Check out the paper to read more about the project.

Brady Boettcher, Johnty Wang, Joseph Malloch, Marcelo M. Wanderley, 2022

Mapper4Live Jam 3/30/22

Jamming with Mapper4Live, a Max4Live device that connects alternate musical controllers to Ableton parameters. A wooden version of the modular musical framework Probatio (Probatio) is used to modulate some synthesizer parameters during the session.

Brady Boettcher, Paul Buser, 2022

TorqueTuner: Rotary haptic force feedback for DMIs

TorqueTuner + Tstick

More detailed demonstration of haptic and audio control mapping using libmapper, SuperCollider, TorqueTuner and the T-Stick DMI.
