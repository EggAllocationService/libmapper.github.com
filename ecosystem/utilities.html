Mapping Utilities

Several useful utilities are included in the repository libmapper-contrib, and more are listed below.

  • mapperRec - Data recording and playback for libmapper.
  • implicitmap - A MaxMSP/Puredata object providing structure for using implicit mapping schemes (e.g. machine learning) with libmapper.
  • OSCmulticast-max-pd - An external object for Max/MSP and Pure Data that can send and receive Open Sound Control over multicast.
