Detailed Description

Databases are the primary interface through which a program may observe the network and store information about devices and signals that are present. Each Database stores records of devices, signals, links, and maps, which can be queried.

◆ mapper_database_subscribe()

void mapper_database_subscribe ( mapper_database  db,
mapper_device  dev,
int  flags,
int  timeout 
)

Subscribe to information about a specific device.

Parameters
dbThe database to use.
devThe device of interest. If NULL the database will automatically subscribe to all discovered devices.
flagsBitflags setting the type of information of interest. Can be a combination of MAPPER_OBJ_DEVICE, MAPPER_OBJ_INPUT_SIGNALS, MAPPER_OBJ_OUTPUT_SIGNALS, MAPPER_OBJ_SIGNALS, MAPPER_OBJ_INCOMING_MAPS, MAPPER_OBJ_OUTGOING_MAPS, MAPPER_OBJ_MAPS, or simply MAPPER_OBJ_ALL for all information.
timeoutThe length in seconds for this subscription. If set to -1, the database will automatically renew the subscription until it is freed or this function is called again.
