Look up the value of a message parameter by symbolic identifier.

Note that it's possible the 'types' string will be longer than the actual contents pointed to; it is up to the usage of this function to ensure it only processes the number of parameters indicated by the 'length' property.

Parameters
msgStructure containing parameter info.
propSymbolic identifier of the property to look for.
Returns
Pointer to mapper_message_atom, or zero if not found.
