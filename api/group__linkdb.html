Look up a link property by index.

To iterate all properties, call this function from index=0, increasing until it returns zero.

Parameters
linkThe link to look at.
indexNumerical index of a link property.
propertyAddress of a string pointer to receive the name of indexed property. May be zero.
typeA pointer to a location to receive the type of the property value. (Required.)
valueA pointer to a location to receive the address of the property's value. (Required.)
lengthA pointer to a location to receive the vector length of the property value. (Required.)
Returns
Zero if found, otherwise non-zero.
