Detailed Description

A monitor may query information about connections between signals on the network through this interface.

It is also used to specify properties during connection requests.

Enumeration Type Documentation

enum _mapper_boundary_action

Describes what happens when the range boundaries are exceeded.

Enumerator:
BA_NONE 

Value is passed through unchanged.

This is the default.

BA_MUTE 

Value is muted.

BA_CLAMP 

Value is limited to the boundary.

BA_FOLD 

Value continues in opposite direction.

BA_WRAP 

Value appears as modulus offset at the opposite boundary.

N_MAPPER_BOUNDARY_ACTIONS 
enum _mapper_instance_allocation_type

Describes the voice-stealing mode for instances.

Enumerator:
IN_UNDEFINED 

Not yet defined.

IN_STEAL_OLDEST 

Steal the oldest instance.

IN_STEAL_NEWEST 

Steal the newest instance.

N_MAPPER_INSTANCE_ALLOCATION_TYPES 
enum _mapper_mode_type

Describes the connection mode.

Enumerator:
MO_UNDEFINED 

Not yet defined.

MO_BYPASS 

Direct throughput.

MO_LINEAR 

Linear scaling.

MO_EXPRESSION 

Expression.

MO_CALIBRATE 

Calibrate to source signal.

MO_REVERSE
