Detailed Description

A device is an entity on the network which has input and/or output signals.

The mapper_device is the primary interface through which a program uses libmapper. A device must have a name, to which a unique ordinal is subsequently appended. It can also be given other user-specified metadata. Devices must be linked before their signals can be connected, which is accomplished by requests from an external GUI.
